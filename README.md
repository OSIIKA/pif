スキルは「準備フェーズに入った瞬間に全艦のスキルが自動発動」する想定です
スキルを発動する順番は、各艦の速力に依存し、速力が最も高い艦から順に発動します
そのために、speedカラムを参照してください
また、同じ速力の艦が複数いた場合は、ランダムに発動順序を決定します
スキルの発動タイミングについては、ターン制を作ってから考えます

まずは、準備フェーズのオーバーレイに、速力の速い順に艦船名、速力、スキル１～３の名前を表示するようにしてください

実際に準備フェーズで使用するためには、戦闘システム全体に影響を及ぼすスキルも追加する必要がある
skillsテーブルは、最低限以下のようなスキルを扱う必要がある
（旗艦・随伴艦全ての）味方艦全体のHPを〇〇％回復する
（旗艦・随伴艦全ての）味方艦のうち最も右側に位置する艦の攻撃力を〇〇％上昇させる
（旗艦・随伴艦全ての）敵艦の攻撃力を〇〇％減少させる
（旗艦・随伴艦全ての）敵艦のHPを〇〇％減少させる
（旗艦・随伴艦全ての）ランダムな味方艦のHPにシールド（付与側のHPの〇〇％）を付与する
